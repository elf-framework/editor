## update ChangeLog

Use changeset to manage changelog. See [here](https://github.com/changesets/changesets/blob/main/docs/intro-to-using-changesets.md) for more details.

```shell
# udpate change log
npm run changeset
```

- Select multiple projects to be changed after executing changeset.
- Then, insert comments for major, minor, and patch.

## update version

Updates the version of package.json for the project in which the changelog is updated.

```shell
npm run version-packages
```
