# Basic 

ELF icon provides material icon wrapped once with sapa library.

## Usage

```jsx {1, 5} showLineNumbers
import AddChartFilled from '@elf-framework/icon/AddChartFilled';
import { start } from "@elf-framework/sapa";

function App() {
    return <AddChartFilled>;
}

start(App);
```

## Category 

You can use the following categories from the list of material icons.

* content
* Android
* editor
* file
* Text Formatting
* Photo&Image
* action
* navigation
* Common actions
