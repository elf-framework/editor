# {frontmatter.title}

Sapa provides Handler as a technology to extend components.

- `StoreHandler` - Handler to deal with MagicMethod for store
- `DomEventHandler` - Handler to deal with MagicMethod for dom event
- `PropsHandler` - Handler to deal with MagicMethod for props
- `ObserverHandler` - Handler to deal with MagicMethod for dom observer

# How to make my own handler

You can make your own handler.

Handler allows components to be extended externally by using three functions: `initialize`, `update`, and `destroy`.

Handler is managed for each component instance.

```jsx
import { BaseHandler } from "@elf-framework/sapa";

registHandler({
  myHandler: class MyHandler extends BaseHandler {
    // mounted
    initialize() {
      // context is a component instance.
      // $el is <div>Simple Text</div>
      console.log(this.context.$el);

      this.context.$el.on("click", (e) => {
        console.log(e);
      });
    }

    // updated
    update() {}

    // destroyed
    destroy() {}
  },
});

function MyComponent() {
  return <div>Simple Text</div>;
}

start(MyComponent);
```
