# {title}

Let's learn the basics of Sapa.
Sapa was created to make UI creation with JavaScript easy.

To use the JS library, you need a few things. Let's find out one by one.

## Import (ESM)

Import the library into your project.

```js
import { start } from "@elf-framework/sapa";
```

## UMD (Using CDN Service)

It can also be used in umd mode.

Let's load the script first.

```html
<script src="https://cdn.jsdelivr.net/npm/@elf-framework/sapa@2.1.23/dist/sapa.umd.min.js"></script>
```

Then use the desired function with the sapa object.

```html
<script>
const { start } = sapa;
</script>
```

# App

The App is the root component of your application.

```js
function App() {
  return (
    <div>
      <h1>Hello World</h1>
    </div>
  );
}
```

## Render

Render the component. 
```jsx
start(App);
```
<Alert type="info" weak={true}>
You can use <strong>render</strong> function instead of{" "}
<strong>start</strong>.
</Alert>

`start` is a function that takes a Sapa component as a parameter.

## VNode

A VNode is a virtual DOM node.

```js
start(<App key="value" />)
```

## Class Component
A class component is a component that is defined in a class.

```js
import { UIElement } from "@elf-framework/sapa";

class App extends UIElement {
  template() {
    return (
      <div>
        <h1>Hello World</h1>
      </div>
    );
  }
}
```

## Function Component

A function component is a component that is defined in a function.

```js
function App() {
  return (
    <div>
      <h1>Hello World</h1>
    </div>
  );
}
```

# RootContext Options 

The `RootContext` is special data that is transmitted only to the root element.

Data for common use in the entire component tree can be injected externally.


## container

The container property renders the component under a specific element.

First, mark the area where the component will be drawn.

```html {5} showLineNumbers
<html>
  ...
  <body>
    <div id="root"></div>
    <script type="module" src="page.jsx"></script>
  </body>
</html>
```

Then you can specify it as a container option.

```jsx {9} showLineNumbers
// @page.jsx
import {start} from "@elf-framework/sapa";

function Main() {
  return <div>MainComponent</div>
}

start(Main, {
  container: document.getElementById('root');
}

```

## using RootContext

`RootContext` is an option given when running start .

To use RootContext inside component, refer [here](/pages/sapa/hook/#useRootContext154).

```jsx {3}
start(<Page {...pageProps} />, {
  pageContext,
  routeParams,
});
```

To use it, do the following:

```jsx {3}
function OtherComponent () {
  const pageContext = useRootContext('pageContext');
  const routeParams = useRootContext('routeParams');

  return <div>Other Component</div>
}
```
