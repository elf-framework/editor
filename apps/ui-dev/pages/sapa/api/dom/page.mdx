# {title}

sapa provides a Dom object for easy manipulation of the dom.

## where is `Dom`

The `Dom` object exists as the root element of all components.

```jsx {3}
function A () {
  return <div onClick={() => {
    console.log(this.$el);
  }}></div>
}
```

## state create(tag, className, attr)

```jsx
const body = Dom.create(document.body);
body.append(document.createElement("div"))

// create with props 
const $el = Dom.create('div', 'add-class my-class', { style: "background-color: yellow" })
```


## static createByHTML(htmlString)

```jsx
const $el = Dom.createByHTML(`
  <div>Simple Text</div>
`);

// or 
const $el = Dom.createByHTML(`
  <svg>
    <rect x="0" y="0" width="100" height="100" />
  </svg>
`);

```


## static createText(text)

```jsx
const text = Dom.createText('text');
```

## static body()

```jsx
// It is same to Dom.create(document.body)
const body = Dom.body();
```

## get tagName()

```jsx
const stringName = $el.tagName
```

## get exists() 

```jsx
const isExists = $el.exists
```

## setAttr(obj)
This is a function to set various properties of the dom object at once.

```jsx
$el.setAttr({
  "data-background-color": "yellow"
})
```

## setAttrNS(obj, namespace = "http://www.w3.org/2000/svg")

Set a property with namespace .

Usually used when creating svg tags.

```jsx
$el.setAttr({
  "data-background-color": "yellow"
}, "http://www.w3.org/2000/svg")
```

## setProp(obj)

Set the property of HTMLElement.

```jsx
$el.setProp({
  value: 10
})
```

## data(key, value)

```jsx
// set data attribute 
$el.data('key', value);

// get data attribute 
const stringValue = $el.data('key');
```

## attr(...args)

```jsx
// set some attribute 
$el.attr('key', value);

// get some attribute 
const stringValue = $el.attr('key');
```

## attrKeyValue(keyField)

```jsx
const div = Dom.createByHTML("<input type='text' value='yellow' data-id='backgroundColor' />");

div.attrKeyValue('data-id') === { 'backgroundColor': 'yellow' }
```

## get attributes()

Brings all attributes of element.

```jsx
const attributes = $el.attributes 
```

See [here](https://developer.mozilla.org/ko/docs/Web/API/Element/attributes) for more details.


## attrs(...args)

```jsx
const div = Dom.createByHTML("<input type='text' value='yellow' data-id='backgroundColor' />");

const [type, value, dataId ] = div.attrs('type', 'value', 'data-id');
```


## styles(...args)

```jsx
const div = Dom.createByHTML("<div style='background-color: yellow; color: blue; border: 1px solid black;'></div>");

const [backgroundColor, color, border ] = div.style('background-color', 'color', 'border');
```
