# Internalization (i18n)

## Introduction

The internalization (i18n) feature allows you to localize your editor. 

## How to use 

### in plugin 

```jsx
function Editor () {
    return <BaseEditor plugins={[
        function (editorContext) {
            const appName = editorContext.i18n.get("app.name");

            console.log(appName);
        }
    ]}>
}
```

### in component

```jsx
function Editor () {
    const { i18n } = useEditor();

    const appName = i18n.get("app.name");

    return <BaseEditor title={appName} />
}
```
