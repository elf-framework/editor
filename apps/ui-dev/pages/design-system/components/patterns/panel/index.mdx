## Options 


### Contents

<OptionsView />

&nbsp;

Content can be expressed freely. If you need a simple message or guide, use a Panel.

  <CodeViewItem title="Sapa">

### Title

<TitleView />

&nbsp;

The panel displays a title with a summary of the area.



<CodeView>
  <CodeViewItem title="Sapa">
```jsx {1} showLineNumbers

    Learn how to work with green screen video with these tips for video
    makers.
</Panel>
```
  </CodeViewItem>
</CodeView>


### Tools 

<ToolView />

The tools is used when adding additional functions other than the title display.


<CodeView>
  <CodeViewItem title="Sapa">
```jsx {3-6} showLineNumbers
<Panel 
    title="Sample Panel" 
    tools={[
        <Button iconOnly quiet>
            <MoreHorizFilled />
        </Button>,
    ]}
>
    Learn how to work with green screen video with these tips for video
    makers.
</Panel>
```
  </CodeViewItem>
</CodeView>


### footer 

<FooterView />

Only standard cards have a footer. The footer has a content area that
can hold buttons on the right side and indicators (badges, icons,
status lights, etc.) on the left. The footer is always divided from
the card body using a small divider.

<CodeView>
  <CodeViewItem title="Sapa">
```jsx {3-6} showLineNumbers
<Panel footer={[
    <Button iconOnly quiet>
        <MoreHorizFilled />
    </Button>,
]}
    title="Sample Panel" 
>
    Learn how to work with green screen video with these tips for video
    makers.
</Panel>
```
  </CodeViewItem>
</CodeView>
