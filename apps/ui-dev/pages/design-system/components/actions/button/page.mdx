# {title}

Buttons allow users to perform an action or to navigate to another page. They have multiple styles for various needs, and are ideal for calling attention to where a user needs to do something in order to move forward in a flow.

## Options 

### Label and icon
<OptionsView />

<CodeView>

<CodeViewItem title="Sapa">
```jsx {5-7} showLineNumbers
import { RoundButton } from "@elf-framework/ui";

function App() { 
    return <Grid columns={[2, 1]} gap={20}>
        <label>With label and icon</label>
        <div>
            <RoundButton variant="primary">
                <FilePresentFilled /> Share
            </RoundButton>
        </div>
        <label>Without icon</label>
        <div>
            <RoundButton variant="primary">
                Share
            </RoundButton>
        </div>
        <label>With icon and label hidden</label>
        <div>
            <IconButton variant="primary">
                <FilePresentFilled />
            </IconButton>

            {/* RoundButton like IconButton */}
            <RoundButton variant="primary" iconOnly>
                <FilePresentFilled />
            </RoundButton>
        </div>
        </Grid>
    
}
```
</CodeViewItem>
    <CodeViewItem title="Html">
    ```html
        <!-- with label and icon -->
        <button class="elf--button primary medium round ">
            <span>{icon} Share</span>
        </button>

        <!-- without icon -->
        <button class="elf--button primary medium round ">
            <span>Share</span>
        </button>

        <!-- with icon and label hidden -->
        <button class="elf--button primary medium round  icon-only">
            <span>{icon}</span>
        </button>
    ```
    </CodeViewItem>

</CodeView>
