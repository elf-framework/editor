# {title}

ELF 는 UI 를 만들기 위해서 기본적으로 2가지 형태로 디자인을 관리한다. 

* design tokens 
* design system 

## Install 

```shell
npm install @elf-framework/design-tokens @elf-framework/design-system
```


## Import 

### JSON data
```jsx
// json 
import tokens from "@elf-framework/design-tokens/tokens.json";
```

### CSS 

```css
@import "@elf-framework/design-tokens/tokens.css";
```
